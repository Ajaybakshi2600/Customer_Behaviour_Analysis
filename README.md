# Customer_Behaviour_Analysis
Data analytics project showcasing customer behavior analysis using Python, SQL and Power BI. 
This project showcases a complete ETL (Extract, Transform, Load) data pipeline, demonstrating proficiency in Python, SQL and Power BI for actionable business intelligence.

Key Highlights: Extraction & Transformation (Python/Pandas): Retrieved raw retail data and performed comprehensive data cleaning, manipulation, and feature engineering within a Jupyter Notebook environment.

Loading & Analysis (MSSQL/SQLAlchemy): Utilized Python's SQLAlchemy library to efficiently load the cleaned data into a MS SQL database.

Business Intelligence (SQL): Executed Exploratory Data Analysis (EDA) and complex queries (including CTEs and Window Functions) in SQL to derive key business insights, such as Customer segmentation, Review ratings and subscription status.\
Visualization & Insights (Power BI): Connect Power BI to SQL server , create new DAX Measures and build an interactive dashboard that highlights key patterns and trends, enabling stakeholders to make data-driven decisions.

This project validates my ability to manage data end-to-end, from raw input to extracting valuable, actionable insights.
